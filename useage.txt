-Add your images to images folder.

-Run the extract_json.py
    The script will extract the file name and save the artist name and image quote in json file with formating.
    File name is numbered sequence.
    Data will be saved in json folder.

-Run rename_images_to_numbered_sequence to rename the images in numbered sequence.

-Create a metadata file for opensea upload using maxime tool. edit the unlockable content section in the generated
json file 

-Paste that file inside Nft_data_for_twitter_bot > collection_scarp_data_from_opensea.json file.

-Run create_data.py.

-Contents are saved in nft_urls_for_twitter.json file.
